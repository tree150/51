以stc89c52rc为芯片，基于普中51A2开发板的菜单系统
使用keil开发平台，利用C语言编写的一个菜单程序。该菜单需要由三个按键（确认健、光标下翻健、返回健）输入控制二级菜单，通过LCD1602显示（下称lcd）。
格式如下：
进入一级菜单，显示三个功能选项：LED、UART、Timer。
左侧有明显的*光标
二、功能要求：
基础部分：
1.	能够正常进入菜单，可执行确认、下翻、返回操作，lcd正常显示。
2.	进入二级菜单，执行二级菜单相关功能。
三、说明
（1）LED
1. 可以使用单片机控制一个LED的亮灭
2. 可以使用PWM波来控制一个LED的亮度，亮度需可调。
（2）UART
1. 使单片机可以向电脑上位机发送任意字符串
2. 使单片机可以接收电脑上位机的数据并正确显示
（3）Timer[LCD实现下列功能]
1. 使定时器中断完成一个计时器系统，从0开始正计时，可暂停
2. 使定时器完成一个简单的时钟系统，包括时/分/秒三项显示并正常运行
